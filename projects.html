2D Particles-SFML

Particle System with SFML A simple particle system built using C++ and the SFML library. This project simulates particles with random positions, velocities, and lifespans, creating a dynamic visual effect. Each particle moves according to its velocity and fades out after a set lifespan, offering a customizable and visually engaging simulation.

Snake-game

This is a classic Snake game developed using SFML (Simple and Fast Multimedia Library). The game features a user-friendly interface with a dynamic menu, allowing players to start or exit the game easily. Players control the snake to collect food while avoiding collisions with the walls and their own tail. The game implements various features, including score tracking, pause functionality, and a visually appealing graphical environment.

SPU-Gradpro

In recent years, Seattle Pacific University's (SPU) enrollment numbers have been decreasing, particularly among students pursuing a Computer Science major. To address this challenge, we, the 3D Trailblazers, developed SPU GradPro, a 3D aided design game aimed at retaining current SPU students in the engineering and computer science department. Our goal is to inspire students to navigate the rigorous curriculum and attract incoming freshmen and transfer students to SPU. The app focuses on concepts from SPU's Data Structures 1 and 2 courses and incorporates LeetCode-inspired questions. Through various engaging game mechanics, SPU GradPro enhances knowledge retention while providing a fun and immersive college experience.
