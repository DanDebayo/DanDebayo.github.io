Projects

This is a curated list of projects that I have worked on throughout the years.

Colorful Fluid Simulations

b'\n

\n An implementation of the classic article Fast Fluid Dynamics Simulation on the GPU \n In C++ and OpenGL. I used the techniques from the article, to implement colorful\n and flashy fluid simulations. I also ran simulations where I used an image of the Mona Lisa as input,\n and this produced quite visually pleasing results.\n A video of this can be seen here.\n

\n '

Hole Fixer

b'\n

\n This is a demo application that demonstrates how to smoothly\n and naturally fill holes in broken 3D meshes. It uses\n techniques from variational calculus and surface fairing to\n achieve this. I have written an article about how it works.\n

\n '

Poisson Blending

b'\n

\n This is my implementation of a technique called Poisson\n Blending. This technique allows you to copy-and-paste one\n image into another, without any ugly seams. In the right\n image, a cute kitten has been copy-and-pasted into a library,\n without any ugly seams whatsoever. I have written an article about how it\n works.\n

\n '

laplacian_deformation.js

b'\n

\n This is my implementation of the paper Laplacian\n Surface Editing. This technique allows you to deform the\n surface of a mesh, while still preserving the details of the\n surface. This is useful when you want to modify a mesh, while\n still making sure that the geometric detail of the mesh is not\n lost. It was implemented using C++ and Eigen, and the C++ code\n was compiled down to WebAssembly, so that it could be used in\n the browser. Click here for a video\n

\n '

Hair Rendering

b'\n

\n A hobby project where I am attempting to render beautiful\n hair in real-time framerates. In the image, the shading model used is\n Kajiya-Kay, and the self-shadows are rendered using Opacity\n Shadow Maps. The project is still a work in progress, and I\n have not yet published the source code.\n

\n '

Reflections using Light Field Probes

b'\n

\n A hobby project where I am attempting to render reflections\n using a field of light probes, inspired by the paper Real-Time\n Global Illumination using Precomputed Light Field\n Probes. I am hoping to use the technique for rendering\n reflections from light probes that are of higher quality than\n parallax corrected environment maps. I also think that it may\n be usable as a better alternative to screen space reflections,\n and it would further allow reflections from off-screen\n objects, which SSR is not capable of.\n

\n '
